当你使用Claude code时，是不是遇到有多个模型需要切换，但需要每个模型的环境变量和配置信息复制粘贴，再启动Claude code
本程序就是解决这个问题，快速切换不同模型对应的环境变量和配置信息，快速启动Claude code。而且还进行了各个模型的信息管理，包括新建、编辑、删除等选项
另外，如果在vscode中需要使用插件Claude code而非cli对话框，本程序还提供了修改settings.json的方法，可以转换配置文件信息到settings.json中，并且可以重置该文件。
本程序为谷歌gemini cli生成，我本人没有编写一句代码。
